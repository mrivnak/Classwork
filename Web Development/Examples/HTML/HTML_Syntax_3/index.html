Course Description

This course is designed to cover a broad range of topics required for modern web application development. It is suitable for intermediate and upper-level computer science students. Topics include: how the Web works, front-end development (HTML/CSS/JavaScript), back-end development (Web Server/PHP/MySQL), advanced development tools, etc.

Course Learning Outcomes

At the completion of this course, the student should be able to:

  • Describe the fundamental knowledge of web development
  • Identify real-life issues and propose a few web solutions for them
  • Design and implement a web application to realize one of the solutions
  • Demonstrate the ability to run and manage a web app
